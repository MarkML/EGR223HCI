HTML Table Exercise

Create a HTML table to display information about fruits.

Fruit Color Taste
Apple Red/Green Sweet
Banana Yellow Sweet
Orange Orange Tangy

Explanation:

  • <table> element is used to create a table.
  • <thead> is the table header section for column headings.
  • <th> defines table header cells.
  • <tbody> is the table body section for data.
  • <tr> defines table rows.
  • <td> defines table data cells.
  • border="1" adds a border to the table for visibility (not recommended in production).

Your Task

Add one more row to the table with information about your favorite fruit.
